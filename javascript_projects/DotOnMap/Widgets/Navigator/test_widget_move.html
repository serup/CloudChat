Animate Curtain Widget

This is an example of how a widget can be moved using JavaScript move.min.js, extremely simple and elegant.

Examples

Move#add/sub(prop, val)

Set prop to val.

Play 
          
	  if(curtain!="on"){
	          	move('#example-1 .box')
         		.add('margin-left', 200)
			.end();
			curtain="on";
	  }
	  else {
	          	move('#example-1 .box')
         		.sub('margin-left', 200)
			.end();
			curtain="off";
	  }
