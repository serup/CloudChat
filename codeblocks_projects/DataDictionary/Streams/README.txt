 Here are descriptions in xml format about the various data flow streams
Each stream is described here

Each stream attribut has detailed info about it, and that detailed info follows
following format:

    <DDEntry>
        <DDGuid></DDGuid>
        <PhysicalDataElementName></PhysicalDataElementName>
        <Location></Location>
        <Category></Category>
        <DataType></DataType>
        <MaxLength></MaxLength>
        <Description></Description>
        <characteristics></characteristics>
        <relationship></relationship>
        <Mandatory>YES</Mandatory>
        <accessrights></accessrights>   
    </DDEntry>


The PhysicalDataElementName is the actual name of the attribut used in the code

Detailed information about the streams connections are described in the systemspecs
